# vscode-motivation-plugin

**Автор:** Кононов Никита Сергеевич (М3115)  
**Название плагина:** Motivation Plugin  
**Кратко:** добрый плагин-мотиватор для VS Code. Похвалит за набор кода, сохранения, фокус без пауз и аккуратные безошибочные сохранения. Показывает прогресс за сессию и мини-стату в статус-баре.

---

## Возможности
- Сообщения каждые N набранных символов (по умолчанию 100).
- Сообщение при сохранении с показом сколько строк добавилось/удалилось.
- Похвала за **первое сохранение после простоя**.
- Похвала за **фокус-сессию** (непрерывная работа N минут).
- **Серия “чистых” сохранений** (без ошибок в Diagnostics) + мини-ачивки.
- Статистика за сессию в статус-баре и командой `Motivation: Show Session Stats`.
- Темы сообщений легко настраиваются в `src/messages.ts`.
